# 特征分类与提问词表

## 1. 用途

本文件同时承担两项职责：

- 提供第二步审核使用的稳定特征中类
- 提供围绕“当前工序在什么条件下存在/不存在”的标准问法

它强调规则可用和提问可用，不追求覆盖底层特征库的全部细名称。

## 2. 三层分类口径

### 2.1 用户可见层

适合页面展示和用户回答的问题词：

- 外圆
- 端面
- 内孔
- 中心孔
- 槽类
- 螺纹
- 倒角或倒圆
- 台阶
- 平面或侧壁
- 曲面
- 小孔或深长孔

### 2.2 审核提问层

适合第二步问答的中类词：

- 外圆类
- 端面类
- 普通孔类
- 中心孔类
- 阶梯孔或孔台阶类
- 回转槽类
- 直槽或键槽类
- 凹槽或通槽类
- 复杂槽类
- 螺纹类
- 平面或侧壁类
- 台阶类
- 倒角倒圆类
- 小孔或深长孔类

### 2.3 内部规则层

适合系统内部理解和归并的流程特征类：

- 外圆柱面
- 轴端面
- 孔
- 阶梯孔
- 孔台阶
- 中心孔
- 环槽
- U形直槽
- 凹槽
- V形直槽或齿槽
- 平面
- 单方向曲面
- 内螺纹或外螺纹

说明：

- 配套内孔、配套外圆、高精度内孔等概念不归入结构特征层，而应归入精度或配套要求层
- 底层细特征可以单独维护，不建议直接进入默认问答文案

## 3. 推荐结构中类

系统在第二步提问时，优先使用以下中类，而不是数据库细特征名：

- 外圆类
- 端面类
- 普通孔类
- 中心孔类
- 阶梯孔或孔台阶类
- 回转槽类
- 直槽或键槽类
- 凹槽或通槽类
- 复杂槽类
- 螺纹类
- 平面或侧壁类
- 台阶类
- 倒角倒圆类
- 小孔或深长孔类

## 4. 提问总原则

### 4.1 优先使用中类词

默认优先用用户能理解、又能落成规则的中类词，不直接使用底层细特征名称。

### 4.2 推荐问题顺序

推荐按三层追问：

1. 先判断是否为命名差异、记录差异，或局部与整体 / 包含关系问题
2. 再进入归因问题
3. 中类确认问题
4. 候选值选择问题

补充原则：

- 不是所有工序都必须直接进入材料、结构、精度、热处理这几类归因。
- 若当前问题本质上只是别名混用、统计口径不同、文档省略或上位工序包含，应优先沿该方向提问，不强行转成工艺因素问题。

### 4.3 候选项优先级

候选项来源优先级：

1. 当前上传文档中真实出现过的值
2. 当前知识包中的默认候选值
3. 其他、手动补充或暂不确定

## 5. 标准归因问法

这类问题用于先判断当前工序主要受什么因素驱动：

- 这道工序是否主要由材料相关因素触发？
- 这道工序是否主要由热处理相关因素触发？
- 这道工序是否主要由结构特征触发？
- 这道工序是否主要由精度或配套要求触发？
- 这道工序是否主要由加工可行性限制触发？

推荐归因选项标准集：

- 材料相关
- 热处理相关
- 结构相关
- 精度或配套要求相关
- 可加工性相关
- 命名或记录方式相关
- 局部与整体 / 包含关系相关
- 暂不确定

## 6. 分类追问词表

### 6.1 结构类

标准问法：

- 这道工序更接近由哪类结构触发？
- 当前结构更接近以下哪一类？

二级追问示例：

- 这个槽更接近回转槽，还是直槽或键槽类？
- 这个孔更接近普通孔、中心孔，还是阶梯孔或孔台阶？
- 是否属于深长孔或小孔类结构？

### 6.2 材料类

标准问法：

- 这道工序主要与哪种材料有关？
- 当前工序的出现是否只在特定材料下成立？

约束：

- 不应默认编造材料候选项
- 文档里材料信息不清时，应允许暂不确定

### 6.3 热处理类

标准问法：

- 这道工序主要与哪种热处理或表面处理状态有关？
- 当前工序是否只在热处理后才需要出现？

### 6.4 精度与配套要求类

标准问法：

- 这道工序是否由更高的精度要求触发？
- 这道工序是否由配套外圆或配套内孔要求触发？
- 前一道精加工是否已经足够，还是仍需继续精整？

### 6.5 可加工性类

标准问法：

- 这道工序是否因为常规方法难以直接完成而增加？
- 当前结构是否需要借助特殊加工方式完成？

## 7. 工序定向问法建议

### 7.1 磨外圆或研外圆

优先问：

- 这道工序是否由配套外圆精度触发？
- 这道工序是否由热处理后的最终定尺寸要求触发？

### 7.2 珩孔或研孔

优先问：

- 这道工序是否由配套内孔精度触发？
- 当前孔精度是否仍无法由前一道工序满足？

### 7.3 磨槽

优先问：

- 这道工序是否由高精度槽或密封槽要求触发？

### 7.4 去应力

优先问：

- 这道工序是否由变形风险或长径比因素触发？

### 7.5 冲洗

优先问：

- 这道工序是否因为存在深长孔结构而需要？

### 7.6 线切割或电火花

优先问：

- 这道工序是否因为存在型孔或特殊开口而需要？
- 这道工序是否因为常规加工方式难以直接完成而增加？

## 8. 不推荐直接使用的问法

以下问法不推荐直接作为第二步默认问题：

- 这个零件还有哪些特征？
- 这个槽到底是不是某种底层细分类槽？
- 这道工序是不是和材料、结构、精度都有关系？

原因：

- 过于发散
- 过于底层
- 用户难以稳定回答
- 不利于系统直接落规则

## 9. 输出导向要求

第二步提问最终应服务规则表达，因此优先追求能收敛出以下类型的答案：

- 某工序主要由材料因素触发
- 某工序主要由热处理状态触发
- 某工序主要由某类结构中类触发
- 某工序主要由配套精度要求触发
- 某工序属于特殊加工可行性驱动工序

如果用户回答后仍不能稳定落规则，应把该结果保留为待确认边界，而不是强行判稳。
