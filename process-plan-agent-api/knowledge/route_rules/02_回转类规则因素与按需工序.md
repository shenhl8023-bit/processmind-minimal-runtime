# 回转类层：规则因素与按需工序

## 1. 适用范围

本层知识适用于以回转表面为主的零件，包括但不限于：

- 轴类零件
- 轴套类零件
- 衬套类零件
- 以外圆、端面、内孔为主的回转件

## 2. 回转类零件的核心加工理解

回转类零件的主加工表面通常包括：

- 外圆
- 端面
- 内孔
- 回转槽
- 螺纹

因此系统在判断路线时，应优先关注：

- 外圆精加工链
- 内孔精加工链
- 回转槽链
- 倒角、去毛刺、清洗、检验等伴随链

## 3. 高频规则因素

### 3.1 配套外圆或高精度外圆

常见影响：

- 触发磨外圆
- 触发研外圆
- 触发热处理后再精加工

### 3.2 配套内孔或高精度内孔

常见影响：

- 触发珩孔
- 触发研孔
- 触发孔精加工链的继续延伸

### 3.3 回转槽或密封槽

常见影响：

- 触发车槽
- 对高精度槽触发磨槽

系统应区分：

- 一般存在槽
- 高精度槽或密封槽

### 3.4 中心孔

常见影响：

- 触发中心孔加工
- 影响后续装夹与基准建立

### 3.5 长径比和变形风险

常见影响：

- 触发去应力
- 影响热处理后精加工链安排

## 4. 常见按需工序理解

### 4.1 磨槽

- 往往由密封槽或高精度槽触发
- 不是所有槽类都需要磨槽

### 4.2 去应力

- 往往与细长件、热处理变形风险相关

### 4.3 冲洗

- 往往与深长孔相关

### 4.4 珩孔 / 研孔 / 研外圆

- 这些工序通常不是因为“有孔/有外圆”而出现
- 更常见的触发因素是精度、粗糙度或配套要求不足

## 5. 补充工艺机理边界

以下内容用于补充系统对常见工艺机理的理解边界：

- 珩磨、铰孔、镗孔都属于已有孔基础上的进一步精整链，不应和起孔动作混同
- 若路线中出现钻后镗、钻后铰，通常意味着对孔精度、圆度或位置度有更高要求
- 渗氮属于应单独识别的热处理因素，其后续精整和检查链不应简单套用一般淬火逻辑
- 深槽、窄槽、端面槽的工艺风险通常更高，槽类问题应继续分支，而不是只问“有没有槽”

## 6. 使用边界

本层知识适合：

- 解释回转类零件的典型精加工链
- 判断哪些因素更可能触发按需工序
- 给后续问题生成提供领域背景

本层知识不应用来：

- 代替当前项目的具体工艺证据
- 直接推出零件专用的业务规则
