# 专用层：业务场景专用知识

## 1. 适用范围

本层知识主要适用于：

- 钢件类活门
- 钢件类衬套
- 活门/衬套组件
- 有色金属类活门或衬套
- 氮化或氰化类活门

如果当前项目并不属于这些零件族，本层知识应只作弱参考。

## 2. 当前业务场景的专用规则因素

### 2.1 配套外圆与配套内孔

这是当前业务场景里非常关键的规则因素。

系统应理解：

- 配套外圆精度会强烈影响磨外圆、研外圆是否出现
- 配套内孔精度会强烈影响珩孔、研孔是否出现
- 前一道精加工若已满足要求，后一道更高精度工序可不出现

### 2.2 锐边质量

在活门和衬套场景中，锐边质量要求明显高于一般回转件。

系统应理解：

- 锐边从最初成形工序开始就需要保护
- 去毛刺、磨工、检验等后续工序都可能受锐边要求约束
- 某些表面处理或光整类工序会被锐边要求抑制

### 2.3 小孔、流量控制孔、深长孔

系统应理解：

- 流量控制小孔、深长孔会带来专门的钻孔、清洗、冲洗、去毛刺要求
- 深长孔结构是冲洗工序的重要触发因素

### 2.4 密封槽和高精度槽

系统应理解：

- 一般槽类通常只触发车槽或常规槽加工
- 密封槽精度要求高时，才更可能进一步触发磨槽

### 2.5 热处理变形风险

系统应理解：

- 钢件淬火、渗氮后更可能引入去应力、时效、后续精磨、探伤检查

## 3. 当前业务场景的专用工艺链

### 3.1 钢件淬火或调质链

常见表现：

- 调质
- 粗加工
- 淬火
- 半精加工
- 精加工
- 磁粉或烧伤检查

### 3.2 渗氮或氰化链

常见表现：

- 正常化
- 粗加工
- 镀铜
- 渗氮或氰化
- 除铜
- 半精加工
- 裂纹或烧伤检查

### 3.3 钝化链

常见表现：

- 精加工或定尺寸完成
- 钝化
- 检验

## 4. 使用边界

本层知识价值很高，但针对性也最强。

它适合：

- 提高专用场景下的规则判断准确性
- 减少按需工序误判
- 帮助系统理解内部典型路线

它不适合：

- 直接外推到所有非回转类零件
- 直接当作全通用机械加工常识
- 压过当前项目正文证据
